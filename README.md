# Lottery Number Generator

A simple web-based lottery number generator with support for Light and Dark modes.

## Deployment
The project is deployed and accessible at:
[https://vialocal1.pages.dev/](https://vialocal1.pages.dev/)

## Features
- Generate 6 random lottery numbers (1-45).
- Dark/Light mode toggle with persistence (localStorage).
- Responsive design.

## How to Run Locally
1. Clone the repository.
2. Open `index.html` in your browser.
